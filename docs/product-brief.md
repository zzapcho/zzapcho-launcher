# 제품 개요

## 제품명

- 한국어 이름: 잡초 약탈서버 런처
- 내부 이름: Zzapcho Raid Launcher

## 목표

`online.zzapcho.kr` 서버에 접속하는 사용자가 공식 런처를 통해 안정적으로 클라이언트를 준비하고 입장하도록 돕는다.

1번 단계의 목표는 실제 제품 개발의 첫 기반을 만드는 것이다.

- 앱이 켜진다.
- 화면 전환이 된다.
- 서버 상태가 보인다.
- 설정이 저장된다.
- 로그를 볼 수 있다.
- 이후 단계가 들어올 구조가 준비된다.

## 대상 사용자

- Windows PC에서 잡초 약탈서버에 접속하려는 일반 사용자
- Java, Minecraft 설치 구조, 모드 파일 위치를 잘 몰라도 접속하고 싶은 사용자

## 제품 톤

차분하고 고급스러운 런처를 목표로 한다.

- 프리미엄
- 미니멀
- 조용함
- 깔끔함
- 과장 없는 안내

피해야 할 것:

- 네온 RGB 게임 런처 느낌
- 사이버펑크 스타일
- AI 생성 이미지처럼 보이는 장식
- 위협적인 보안 문구
- 복잡한 설정 노출

## 고정 서버 정책

런처는 `online.zzapcho.kr:25565` 전용이다.

사용자가 서버 주소를 직접 추가하거나 수정하는 기능은 제공하지 않는다.
다른 서버 접속 제한은 최종적으로 서버 플러그인과 백엔드에서 처리해야 한다.
